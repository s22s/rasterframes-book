    "# Getting&nbsp;Started\n",
    "\n",
    "## Quick Start\n",
    "\n",
    "RasterFrames is available in in a Jupyter Notebook Docker container for quick experimentation.\n",
    "\n",
    "1. Install [Docker](https://www.docker.com/get-docker) for your OS flavor.\n",
    "2. Run  \n",
    "   `docker run -it --rm -p 8888:8888 -p 4040-4044:4040-4044 s22s/rasterframes-notebooks`\n",
    "\n",
    "Additional instructions can be found [here](https://github.com/locationtech/rasterframes/blob/develop/deployment/README.md).\n",
    "\n",
    "## General Setup\n",
    "\n",
    "*RasterFrames* is published via [Maven Central](http://search.maven.org/#search%7Cga%7C1%7Crasterframes) (click link to see latest versions).\n",
    "\n",
    "To use RasterFrames, add the following library dependencies:\n",
    "\n",
    "`dependency[sbt,Maven,Gradle] {\n",
    "  group=\"io.astraea\"\n",
    "  artifact=\"rasterframes_2.11\"\n",
    "  version=\"x.y.z\"\n",
    "}`\n",
    "\n",
    "`dependency[sbt,Maven,Gradle] {\n",
    "  group=\"io.astraea\"\n",
    "  artifact=\"rasterframes-datasource_2.11\"\n",
    "  version=\"x.y.z\"\n",
    "}`\n",
    "\n",
    "Optional:\n",
    "\n",
    "`dependency[sbt,Maven,Gradle] {\n",
    "  group=\"io.astraea\"\n",
    "  artifact=\"rasterframes-experimental_2.11\"\n",
    "  version=\"x.y.z\"\n",
    "}`\n",
    "\n",
    "It assumes that SparkSQL 2.2.x is available in the runtime classpath. Here's how to add it explicitly:\n",
    "\n",
    "`dependency[sbt,Maven,Gradle] {\n",
    "  group=\"org.apache.spark\"\n",
    "  artifact=\"spark-sql\"\n",
    "  version=\"2.2.1\"\n",
    "}`\n",
    "\n",
    "Note:\n",
    "Most of the following examples are shown using the Spark DataFrames API. However, many could also be rewritten to use the Spark SQL API instead. We hope to add more examples in that form in the future.\n"
   ]
